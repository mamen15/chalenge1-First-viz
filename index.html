Age Distribution Chart

School Type and Area Distribution

Student Performance

This radar chart provides an overview of individual student performance in various subjects. The chart includes scores in primary, secondary, and current education levels, as well as scores in mathematics, Arabic language, and the first language. Each axis represents a different subject area, and the data points indicate the corresponding scores for each student. Please select a specific student from the dropdown menu ("Select Student") to display their performance data.
